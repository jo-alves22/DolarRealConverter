# Conversão de Moedas

Este é um aplicativo desenvolvido em .NET MAUI que permite a conversão de valores entre Dólar (USD) e Real (BRL) e vice-versa. O aplicativo consulta automaticamente a cotação atual do Dólar em relação ao Real, permitindo que o usuário faça conversões precisas.

## Funcionalidades

- **Conversão de Dólar para Real**
- **Conversão de Real para Dólar**
- **Busca automática da cotação atual do Dólar através de uma API HTTP**

## Tecnologias Utilizadas

- **.NET MAUI**: Framework utilizado para o desenvolvimento multiplataforma do aplicativo.
- **HTTP Client**: Biblioteca utilizada para buscar o valor da cotação atual da moeda.

## Como Executar

1. Clone este repositório.
2. Abra o projeto no Visual Studio.
3. Compile e execute o aplicativo em um emulador ou dispositivo real.
